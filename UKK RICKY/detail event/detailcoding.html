Tentang Event

Informasi lengkap event hackathon

Mini Hackathon adalah event coding khusus untuk para developer, UI/UX designer, dan programmer yang ingin berkolaborasi membangun project kreatif dalam waktu terbatas.

Peserta akan bekerja dalam tim untuk menciptakan solusi digital, aplikasi web, ataupun mobile app dengan tema teknologi masa depan.
